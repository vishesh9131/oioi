![安装选项](tutorial_assets/1.png)

## 卸载 oioi 🗑️
1. 从菜单栏退出 oioi
2. 从应用程序文件夹拖到垃圾桶
3. 删除数据文件夹：
   ```bash
   rm -rf ~/Library/Application\ Support/oioi
   ```
